# Task 5: Capture and Analyze Network Traffic Using Wireshark

## Objective
To capture live network traffic and identify basic protocols using Wireshark.

## Tool Used
Wireshark

## Steps Performed
1. Installed Wireshark
2. Selected active Wi-Fi interface
3. Started packet capture
4. Generated traffic by browsing websites
5. Stopped capture after one minute
6. Identified protocols in capture
7. Saved capture file as .pcap

## Protocols Identified
- DNS (Domain Name System)
- QUIC (Encrypted transport protocol)
- ICMPv6 (Network control messages)

## Outcome
Learned how to capture live network traffic and identify different network protocols.
